# tools

Dev tools that help the team, not the player. Nothing here ships with the game.

Examples of what belongs here:

- Level editors or procedural generation scripts
- Asset import/export helpers (batch renaming, atlas packing, etc.)
- Build scripts and CI helpers
- Data validation or balancing spreadsheets
- Profiling or optimization utilities

If a tool is useful across multiple Patate projects, consider contributing it upstream.
